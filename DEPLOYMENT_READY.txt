✨ STEP 1: PUSH KE GITHUB (5 MENIT)

Opsi A - Interactive Script (RECOMMENDED):

$ bash PUSH_TO_GITHUB.sh

Opsi B - Manual Command:
$ git remote add origin https://github.com/YOUR_USERNAME/elc-baru.git
$ git push -u origin main

✨ STEP 2: CONNECT CLOUDFLARE PAGES (10 MENIT)
───────────────────────────────────────────────

1. Go to: https://dash.cloudflare.com/
2. Sidebar > Pages > Create a project
3. Connect to Git > Authorize GitHub > Select 'elc-baru'
4. Build Settings:
   - Production branch: main
   - Framework: Next.js
   - Build command: npm run build
   - Output directory: .next
5. Click: Save and Deploy
6. Wait for deployment (2-5 minutes)
7. Your site will be live at: https://elc-baru.pages.dev

✨ STEP 3: SETUP CUSTOM DOMAIN (15 MENIT - OPTIONAL)
────────────────────────────────────────────────────

1. Pages Project > Settings > Custom domain
2. Add domain: www.edelweisslearning.id
3. Update DNS with provided CNAME record
4. Wait for DNS propagation (5-48 hours)
5. SSL certificate auto-generates (15-30 min)

✓ TOTAL TIME: ~30-40 minutes

📞 SUPPORT & TROUBLESHOOTING

✓ Build failed: Check .next output directory
✓ Domain not working: Wait for DNS propagation
✓ Deploy failing: Check environment variables
